# AI Resume Matcher 🚀

An AI-powered Resume Analysis and Job Matching System built using **Python, Streamlit, and Google Gemini AI**.

This application analyzes a candidate's resume against a given job description and provides intelligent insights such as resume compatibility score, strengths, missing skills, and improvement recommendations.

The system helps candidates optimize their resumes according to specific job requirements.

---

## 📌 Project Overview

In today's competitive job market, candidates often apply for multiple roles without knowing how well their resumes match specific job requirements.

AI Resume Matcher solves this problem by automatically analyzing resumes and providing AI-based feedback.

The application uses Google's Gemini AI model to understand resume content and compare it with job descriptions.

---

# ✨ Features

## Resume Analysis

- Upload resume in PDF format
- Extract resume text automatically
- Analyze resume content using AI


## Job Matching

- Enter target job title
- Add job description
- Compare resume with job requirements


## AI Generated Insights

- Resume Match Score
- Resume Strengths
- Missing Skills Identification
- Improvement Suggestions


## AI Chat Assistant

- Ask questions related to resume analysis
- Get personalized AI responses
- Understand improvement areas
